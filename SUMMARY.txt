## 🛠️ EXTENDING WITH NEW RUST MODULES

To add a new Rust feature:

1. Create lyra_core/src/my_module.rs
2. Add to lyra_core/src/lib.rs:
   ```rust
   mod my_module;
   m.add_class::<MyClass>()?;
   ```
3. Rebuild: ./build_lyra_core.sh
4. Use in Python: from lyra_core import MyClass


## ❓ TROUBLESHOOTING

Issue: "ModuleNotFoundError: No module named 'lyra_core'"
→ Run: ./setup.sh

Issue: "No ⚡ indicator on startup"
→ Rust module not loaded. Run: python verify.py

Issue: "Build fails"
→ Check Rust: rustc --version
→ Rebuild: cd lyra_core && cargo clean && maturin develop --release

Issue: "Slow performance"
→ Verify Rust loaded: python -c "from lyra_core import JsonHandler"
